## Файлы ##

- Файловый вариант: База - .1CD файл;
- Серверный вариант: База - Файлы СУБД;
- .cf - это файл конфигурации (только программный код без данных);
- .dt - файл выгрузки базы (код + данные);
- .epf - файл внешней обработки;
- .erf - файл внешнего отчета.
